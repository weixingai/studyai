# ChatGPT Prompt 进阶指南

学习如何编写更好的提示词，获得更准确的回答。

## 什么是 Prompt Engineering？

Prompt Engineering 是优化与 AI 模型交互的提示词的技术。

## 核心原则

1. 明确性
2. 具体性
3. 上下文提供
4. 示例引导

## 高级技巧

- 角色设定
- 格式控制
- 思维链提示
- 迭代优化
